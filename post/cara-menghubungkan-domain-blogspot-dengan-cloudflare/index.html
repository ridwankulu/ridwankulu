Cloudflare merupakan layanan CDN yang sangat powerfull digunakan untuk mengantisipasi serangan DDOS, hack dan lain sebagainya. Salah satu keunggulan dari cloudflare ada menyembunyikan identitas website seperti nameserver atau IP utama. Karena semuanya akan diproses menggunakan cloudflare, sehingga yang akan tampil data dari cloudflare.

Jika saat ini menggunakan blogspot/blogger, tentunya sangat baik jika menggunakan cloudflare. Selain menyembunyikan identitas blogspot, juga dapat mempercepat website. Selain itu, ini juga mungkin akan menjawab pernyataan para pembaca tentang “Bagaimana Cara menghubungkan domain yang kita dapat dari Domainer.co.id ?”. Saya sudah meletakan cara disni. Lalu bagaimana cara menghubungkan domain ke blogspot melalui cloudflare, ini dia :
